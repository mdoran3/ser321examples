You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html

  • /multiply? -- returns the product of two integers.
    Example: mdoran3.duckdns.org/multiply?num1=2&num2=5

  • /github?query=users/{github_username}/repos returns a parsed JSON of all the
    public repositories of the specified github user (ie {github_username})
    Example: mdoran3.duckdns.org/github?query=users/mdoran3/repos

  • /birthday? -- returns the number of days until your next birthday.
    Provide the 'day' you were born and the 'month' you were born in integer form.
    Example: mdoran3.duckdns.org/birthday?day=28&month=4

  • /happyMadison? -- enter 'happy' for 'movie' parameter and an integer 1, 2, or 3 for 'quote' to see one of three famous Happy Gilmore quotes.
    or enter 'madison' for 'movie' parameter and an integer 1, 2, or 3 for 'quote' to see one of threeee famous Billy Madison quotes.
    Example: mdoran3.duckdns.org/happyMadison?movie=happy"e=1
